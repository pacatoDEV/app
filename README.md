# Sonda Service Desk Pro (Uso Simplificado) 🚀

Para que esta ferramenta seja prática para técnicos com restrições de instalação nos computadores da SONDA/DASA, **não é necessário baixar código ou instalar nada**.

## ☁️ Como usar sem instalar nada - A OPÇÃO MAIS PRÁTICA

A aplicação é **Web-First**. Isso significa que ela roda inteiramente no seu navegador de internet, sem precisar de permissões de administrador.

1. **Acesse o link direto:** Utilize o URL gerado na plataforma (Shared App URL).
2. **Barra de Favoritos:** Salve o link nos favoritos (Ctrl+D) do seu Chrome ou Edge para acesso instantâneo.
3. ** Desktop Shortcut (Atalho):** 
   - No Chrome/Edge, clique nos 3 pontos (...) > Salvar e Compartilhar > **Instalar esta página como um aplicativo**. 
   - Isso criará um ícone igual a um programa no seu Desktop, mas ele roda leve e sem precisar de permissões especiais.

---

## 🛠️ Para quem baixou o código (Desenvolvedores)
Se você baixou os arquivos e quer rodar localmente no seu computador pessoal (onde pode instalar coisas):
1. Instale o [Node.js](https://nodejs.org/).
2. Abra o terminal na pasta e use:
   ```bash
   npm install
   npm run dev
   ```

---

## ✨ Por que esta ferramenta é ideal para o SD?
- **Zero Instalação:** Não gasta espaço no disco do PC do trabalho.
- **Privacidade Local:** Seus textos salvos ficam no navegador. Ninguém mais vê o que você salvou na sua sessão.
- **Portabilidade:** Você pode abrir no seu celular pessoal se precisar consultar uma solução rápida fora da máquina.
